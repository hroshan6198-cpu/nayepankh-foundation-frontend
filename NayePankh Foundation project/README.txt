# NayePankh Foundation Front End Project

This is a simple responsive landing page for NayePankh Foundation.

## Features
- Responsive design
- Multiple sections
- Smooth modern UI
- Scroll animations
- Dark mode toggle
- Contact form design

## How to Run
1. Extract the ZIP file.
2. Open `index.html` in any browser.
3. Keep `index.html`, `style.css`, and `script.js` in the same folder.
